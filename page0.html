About This Website

This Student Resource Management System was developed as a full-stack academic project to simplify collaboration between students, teachers, and administrators. The system includes dashboards, activity management, authentication, real-time interactions, database integration, and deployment features.

What I Learned

Through this project, I gained practical experience in both frontend and backend web development while learning real-world deployment workflows.

  • Frontend design using HTML, CSS, and JavaScript
  • Backend logic and authentication using PHP
  • Database management with MySQL
  • Building login and registration systems
  • Working with Socket.IO for communication features
  • Hosting and deployment using InfinityFree and Render
  • Version control using Git and GitHub
  • Creating responsive UI and interactive effects
  • Understanding real-world project structure
